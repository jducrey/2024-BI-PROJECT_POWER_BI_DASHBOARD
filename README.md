# PROJECT-POWER-BI-DASHBOARD
Data Analysis and Data Visualization Project, through the creation of a dashboard, using the Power BI tool.

This Dashboard was developed as part of a personal project, to demonstrate my data visualization skills using the Power BI tool.

The Global Super Store dataset from Kaggle encompasses sales, profit, customer demographics, and product information across various regions, providing insights into retail performance, customer behavior and operational efficiency, on severals differents scales.
This dataset is available on this link:
- https://www.kaggle.com/datasets/ronysoliman/global-superstore-dataset

This dashboard consists of multiple pages, allowing for analysis and visualization of all the informations provided by this dataset.
