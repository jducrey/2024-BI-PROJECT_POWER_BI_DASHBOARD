# 📊 Power BI Dashboard Project
## 🏆 Overview
This project showcases data analysis and visualization through an interactive Power BI dashboard. The goal is to demonstrate my expertise in data visualization by analyzing the Global Super Store dataset from Kaggle.

The dataset includes:
✔ Sales & Profit

✔ Customer Demographics

✔ Product Information

✔ Operational & Logistics Data

## 📂 Dataset available on Kaggle:
🔗 Global Super Store Dataset

The Global Super Store dataset from Kaggle encompasses sales, profit, customer demographics, and product information across various regions, providing insights into retail performance, customer behavior and operational efficiency, on severals differents scales.
